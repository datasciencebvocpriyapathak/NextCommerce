# NextCommerce
A full-stack eCommerce web app built with Next.js, Sanity CMS, and Razorpay — featuring dynamic product management, seamless checkout, and secure payment integration.
A modern, full-stack eCommerce application built with Next.js, Sanity.io, and Razorpay — designed for performance, scalability, and a smooth shopping experience.
✨ Features

⚡ Next.js — Fast, SEO-friendly storefront with server-side rendering and static generation
🗂️ Sanity CMS — Flexible, real-time content management for products, categories, and collections
💳 Razorpay Integration — Secure and seamless payment gateway with support for UPI, cards, netbanking, and wallets
🛒 Cart management with persistent state
📦 Order summary and confirmation flow
📱 Fully responsive design
🔐 Secure checkout with server-side payment verification

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js, Tailwind CSS |
| CMS / Backend | Sanity.io |
| Payments | Razorpay |
| Deployment | Vercel |
